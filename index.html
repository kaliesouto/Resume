My Recipes Website

This was the final project for my capstone class, INFSCI 1059 - Web Programming course. The project description was to create a website that utilized three services. For my website, I created a site that the user could add websites that contained recipes to a database. Later the user could query the database to find specific recipes based off of names or tags the user attributed to the website initially. The second service the website had was a shopping list. The user could easily add ingredients to the list as they ran out or delete them as they purchased them. Lastly, the third service was a forum. Users could post question or comments and get responses from other users. The forum utilized permission levels, where most users could add topics, reply to posts, and delete their own responses. Users with the admin level of permissions can do anything a regular user can, add a category, and delete categories and any posts. The whole website utilizes a username and password to access it. To access the site you can use 'test' and 'test' as the usernamea and password. The project was mostly focused on the backend development rather than styling, since that is focused on in another course, so it is minimal.
